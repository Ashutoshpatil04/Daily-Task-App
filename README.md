# Daily Tasks - Android App

## Project Structure
- `/www` - Web app files (HTML, CSS, JS)  
- `/android` - Android native project (generated by Capacitor)

## Building the APK

### Prerequisites
1. Node.js installed
2. Android Studio installed (for building APK)

### Build Steps
```bash
# Install dependencies
npm install

# Build for Android
npx cap sync android

# Open in Android Studio to build APK
npx cap open android
```

### Once in Android Studio:
1. Go to Build → Build Bundle(s) / APK(s) → Build APK(s)
2. The APK will be in: android/app/build/outputs/apk/debug/app-debug.apk
3. Transfer this APK to your phone and install it

## Data Storage
All tasks are stored locally on your device using the device's local storage.
No internet connection required after installation.
